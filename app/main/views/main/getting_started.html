Overview

Volt is a Ruby web framework where your Ruby code runs both on the server and the client (via opal.) The DOM automatically updates as the user interacts with the page. Page state can be stored in the URL. If the user hits a URL directly, the HTML will first be rendered on the server for faster loading and easier indexing by search engines.

Instead of syncing data between the client and the server via HTTP, Volt uses a persistent connection between the client and the server. When data is updated on one client, it is updated in the database and any other listening clients (with almost no setup code needed).

HTML for pages is written in a Handlebars-like template language. Volt uses data flow/reactive programming to automatically and intelligently propagate changes to the DOM (or any other code wanting to know when a value has been updated). When something in the DOM changes, Volt intelligently updates only the nodes that need to be changed.
